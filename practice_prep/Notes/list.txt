q:explain shallow copy vs deep copt for lists?
ans: shallow copy creats a new list but nested objects are still referenced.
deep copy creats completely independent copies of all nested objects.

import copy
a = [[1,2],[3,4]]
b = a.copy() # shallow
b[0][0] = 99 # also chanes a!


c=copy.deepcopy(a) # deep copy
c[0][0] = 88 #a is unaffected

Q1: how does memory management differ between a list and tuple?
answer: lists are mutable and over-allocate memory to allow  for efficient append operations(dynamic arrays)
tuples are immutable and fixed-size, making them more memory-dffient and afaster to iterate over.

Ams twist: "Because tuples are immutable, python can optimize them by caching small tuples in a resource pool, reducing memory fragmentation"

Q: how is a python dictionary implemented internally ?
ans: it is implemented as a hashtable(hash map). it uses the hash() of the keyto calculate the index in an array (bucket) where the value is stored
complexity: average case for lookup, iszert, and delete is O(1). worst case (collisions) is O(n).
